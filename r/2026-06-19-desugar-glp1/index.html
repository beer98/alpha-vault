01诊断 · 这到底是什么趋势

不是减肥潮，是一次需求的大迁移

先看两个"全社会都知道、但没人当成投资"的事件。它们本身就是趋势已经势不可挡的证据。

真正的洞察 · 把营销标签撕掉，读一级机制

GLP-1 不是减肥药。它作用在多巴胺奖赏回路上——是人类第一种可规模化的"反冲动"技术。

而现代消费经济很大一块，本质是靠"人停不下来"赚钱。这味药一来，那个驱动不会消失——钱会从"量"大规模迁移到"质、外形、健康"。所以问题不是"什么会死"，而是被释放的钱包流向哪、谁拥有那个新对象。

02方法 · 怎么挖到能买的东西

别买药。药会变成 DRAM。

减肥药从发明那天起，利润就是"高位窗口 → 专利到期 → 仿制 → 价格战 → 商品化"的注定衰减曲线（中国替尔泊肽已经打到两折）。要买的是不管谁打价格战都得向它交钱、又没有专利悬崖的东西。

在内存周期里，你不持有做内存的，你持有 ASML——那个谁都得向它买、没有专利悬崖的卖铲人。

把这个逻辑套到去糖化，耐久的钱聚在三个地方，而不在最显眼的药身上：

A

药亲手造的下一个市场

它造出松弛皮肤 → 医美塑形；它造出肌肉流失 → "陪它一起吃"的保肌药。趋势递归地制造自己的需求。

B

卖铲人 / 卡脖子

乳清蛋白、玻璃管、注射笔、灌装产能——按"剂/人"收过路费，谁赢都收钱，没有专利悬崖。

C

清洁版替代

被压制的瘾会迁移：酒精→无醇/THC/适应原饮料。香烟→Zyn 的剧本重演。但好品牌多在私募。
